# フォールバックテーマ（tech-scrapbook が使えない環境向け）

`theme: tech-scrapbook` が使えない場合は以下のフロントマターを使用する。

```markdown
---
marp: true
theme: default
paginate: true
header: "Tech Scrapbook"
style: |
  section { font-size: 28px; }
  table { font-size: 0.78em; }
  pre { margin: 0.4em 0; }
  pre code { font-size: 0.7em; line-height: 1.4; }
  h2 { margin-bottom: 0.3em; }
  h3 { margin-top: 0.5em; margin-bottom: 0.2em; }
  ul, ol { margin: 0.3em 0; }
  p { margin: 0.3em 0; }
---
```
